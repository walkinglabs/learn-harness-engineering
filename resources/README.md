# Resource Library

This folder is the bilingual starter library for the course. It turns the
course methods into copy-ready templates and compact references you can use in a
real repository.

## Choose A Language

- 中文资料库: [`zh/README.md`](./zh/README.md)
- English library: [`en/README.md`](./en/README.md)

## What You Will Find

- root instruction templates
- feature and progress state files
- handoff and clean-exit checklists
- bootstrap script references
- compact method notes for initializer and coding-agent flows

## Recommended Starter Pack

- `AGENTS.md` or `CLAUDE.md`
- `feature_list.json`
- `claude-progress.md`
- `init.sh`

Start with those four files first, then add handoff and evaluator artifacts as
the project becomes longer-running or higher-risk.
