# Lecture 01. Why Capable Agents Still Fail

## Guiding Question

Why do capable models still fail on real software tasks?

## Why This Matters

Harness engineering begins when model capability is no longer the main
bottleneck. Strong agents still fail when their environment is underspecified,
their goals are vague, and their feedback loops are weak.

## Core Ideas

- Reliability is a systems property, not just a model property.
- Many failures that look like model weakness are actually harness weakness.
- When an agent struggles, the useful question is often: what was missing from
  the environment?
