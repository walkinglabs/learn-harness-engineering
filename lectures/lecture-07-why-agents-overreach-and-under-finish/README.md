# Lecture 07. Why Scope Must Be Made Explicit

## Guiding Question

Why do agents fail when scope is vague, oversized, or implicit?

## Why This Matters

Agents struggle when they cannot see the real boundary of the task. Harnesses
improve reliability by making scope explicit, enumerable, and incremental.

## Core Ideas

- Agents tend toward one-shot behavior when scope is not externalized.
- Large, vague tasks increase drift and false completion.
- Feature lists turn intent into a visible work surface.
- Incremental progress is a harness design decision, not a model trait.
