# Lecture 11. Why Advanced Harnesses Split Roles

## Guiding Question

Why do stronger harnesses separate planning, implementation, and evaluation?

## Why This Matters

As tasks become broader and quality standards rise, a single agent tends to
overload itself. Advanced harnesses split roles so different failure modes can
be handled explicitly.

## Core Ideas

- Self-evaluation is systematically weak.
- Evaluators provide skepticism and back-pressure.
- Planners help expand and structure scope before implementation begins.
- More complex harnesses are justified only when those roles are load-bearing.
